## How to read this portfolio

- **If you have ten minutes** → read the project READMEs and skim the [`sample_outputs`](./project1_sql_ecommerce/sample_outputs/) folder of Project 1 and the [`outputs`](./project2_python_sales/outputs/) folder of Project 2. Every chart and CSV is pre-generated.
- **If you want to verify the code runs** → each project has its own `requirements.txt` and a one-line "How to run" section. Project 1 also runs without PostgreSQL via the bundled SQLite script.
- **If you only have time for one thing** → look at Project 1's `queries.sql` to gauge SQL depth, or Project 3's dashboard once it lands.
